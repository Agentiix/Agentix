---
title: Integrate an agent
description: Wrap an agent CLI (Claude Code, Aider, OpenHands, Codex ...) as a Python module.
---

An **agent** in Agentix is a Python module whose function shells out to
a CLI binary. The caller imports that function and invokes
`c.remote(run, ...)`; the
body, running inside the sandbox, execs the binary and returns a typed
result.

This guide ports `claude-code` as the worked example.

## 1. Source layout

```text
agentix-claude-code/
├── pyproject.toml
├── default.nix              # Nix derivation — pulls the claude CLI binary
└── src/agentix/
    └── claude_code/
        └── __init__.py      # async def run(...), types
```

## 2. Method surface

```python title="src/agentix/claude_code/__init__.py"
from __future__ import annotations
import asyncio
import os
from dataclasses import dataclass


@dataclass
class RunResult:
    exit_code: int
    stdout: str
    stderr: str


async def run(
    instruction: str,
    *,
    workdir: str = "/testbed",
    timeout: float = 600,
    model: str | None = None,
    max_turns: int | None = None,
    env: dict[str, str] | None = None,
) -> RunResult:
    cmd = ["claude", "-p", instruction, "--print", "--permission-mode", "bypassPermissions"]
    if model:
        cmd += ["--model", model]
    if max_turns is not None:
        cmd += ["--max-turns", str(max_turns)]

    proc = await asyncio.create_subprocess_exec(
        *cmd,
        cwd=workdir,
        env={**os.environ, **(env or {})},
        stdout=asyncio.subprocess.PIPE,
        stderr=asyncio.subprocess.PIPE,
    )
    try:
        stdout, stderr = await asyncio.wait_for(proc.communicate(), timeout=timeout)
    except TimeoutError:
        proc.kill()
        await proc.communicate()
        return RunResult(exit_code=-1, stdout="", stderr=f"timed out after {timeout}s")

    return RunResult(
        exit_code=proc.returncode or 0,
        stdout=stdout.decode(errors="replace"),
        stderr=stderr.decode(errors="replace"),
    )
```

The function body runs **inside the sandbox** — anything an
`asyncio.create_subprocess_exec` call can do is fair game.

## 3. Packaging

```toml title="pyproject.toml"
[project]
name = "agentix-claude-code"

[tool.hatch.build.targets.wheel]
```

`pip install agentix-claude-code` resolves
`from agentix.claude_code import run` on the caller side.
`agentix build ./agentix-claude-code` produces the image.

## 4. Extract the patch on the caller

Patch extraction is a generic "what changed in this workdir"
operation; keep it on the caller, not in the agent module:

```python
from agentix.bash import run as bash_run

diff = await c.remote(
    bash_run,
    command=f"cd /testbed && git add -A && git diff --cached --no-color",
)
patch = diff.stdout
```

The same call works for any agent that edited files.

## 5. Streaming output

For long-running agents, add a streaming variant. A function annotated
`-> AsyncIterator[Event]` auto-upgrades to Socket.IO:

```python
from collections.abc import AsyncIterator
from dataclasses import dataclass
from typing import Literal


@dataclass
class TokenChunk:
    text: str
    type: Literal["token"] = "token"


@dataclass
class RunDone:
    result: RunResult
    type: Literal["done"] = "done"


async def run_stream(instruction: str, **kwargs) -> AsyncIterator[TokenChunk | RunDone]:
    proc = await asyncio.create_subprocess_exec(
        "claude", "-p", instruction, "--output-format", "stream-json",
        stdout=asyncio.subprocess.PIPE, stderr=asyncio.subprocess.PIPE,
    )
    async for raw in proc.stdout:
        event = json.loads(raw)
        match event["type"]:
            case "token":  yield TokenChunk(text=event["text"])
            case "done":   yield RunDone(result=_build_result(event))
```

Callers iterate without `await`:

```python
from agentix.claude_code import run_stream

async for ev in c.remote(run_stream, instruction="…"):
    match ev:
        case TokenChunk(text=t): print(t, end="")
        case RunDone(result=r):  return r
```

## 6. The Nix derivation

Agent CLIs are usually unpackaged outside Nix. The simplest path:
fetch the upstream tarball, copy the binary, expose it on `bin/`.
See [`agentix-cookbook/claude-code/default.nix`](https://github.com/Agentiix/agentix-cookbook/blob/main/claude-code/default.nix)
for a working reference.

## Secrets and workdir

- Pass API keys per call via `env={"ANTHROPIC_API_KEY": "..."}`. Don't
  bake them into the image.
- Default `workdir=/testbed`. Most benchmarks mount the repo there.
- Bound `timeout` aggressively. Long-running agents can hang.
