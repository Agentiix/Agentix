---
title: Namespace protocol
description: The contract between a namespace, the runtime, and the caller — package layout, entry-point declaration, wire format.
---

A **namespace** is a normal Python distribution that contributes to
the `agentix.*` import namespace and declares one `agentix.namespace`
entry point. Callers invoke `await c.remote(fn, ...)`; the runtime
reads `fn.__module__` as the routing key, imports the package lazily,
and runs the body inside the sandbox.

## Package layout

```text
agentix-myagent/
├── pyproject.toml
└── src/
    └── agentix/              # PEP 420 namespace package — no __init__.py
        └── myagent/
            └── __init__.py   # public async functions
```

The framework's `agentix/__init__.py` extends its `__path__` via
`pkgutil.extend_path`, so any wheel installing files at
`<site-packages>/agentix/myagent/` becomes importable as
`from agentix.myagent import run`.

**Reserved framework subpackages** — namespace dists may not shadow:
`agentix.cli`, `agentix.deployment`, `agentix.dispatch`,
`agentix.idents`, `agentix.models`, `agentix.namespace`,
`agentix.rollout`, `agentix.runtime`, `agentix.trace`. Everything else
under `agentix.*` is fair game.

## The package IS the namespace

```python title="src/agentix/myagent/__init__.py"
async def run(instruction: str) -> str:
    return f"did: {instruction}"
```

- Public async functions (`async def` / `async def ... yield`) are
  remote-callable methods.
- Dataclasses, constants, sync helpers — callers import them as
  regular Python.
- Heavy native dependencies belong in lazy imports inside the function
  body so caller-side `import` stays cheap.

## Entry-point declaration

```toml title="pyproject.toml"
[project]
name = "agentix-myagent"
version = "0.1.0"
requires-python = ">=3.11"

[project.entry-points."agentix.namespace"]
myagent = "agentix.myagent"

[tool.hatch.build.targets.wheel]
packages = ["src/agentix"]
```

The key (`myagent`) is the short name used by `agentix build` and
`agentix check`. The value (`agentix.myagent`) is the import path.

Class-style targets — `myagent = "agentix.myagent:MyClass"` — are
also accepted for discovery.

## Wire

### Unary — `POST /_remote`

```text
POST /_remote
  { "package": "agentix.myagent",
    "method":  "run",
    "args":    [],
    "kwargs":  { "instruction": "fix the bug" } }
```

Success — always 200:

```json
{ "ok": true, "value": "did: fix the bug" }
```

Failure — also 200, with the exception class in the payload:

```json
{ "ok": false, "error": { "type": "ValueError", "message": "...", "traceback": "..." } }
```

Only "package not found in registry" returns 404. W3C `traceparent`
and `tracestate` headers pass through.

### Streaming, bidirectional, log subscription — Socket.IO

`/socket.io/` carries every non-unary call. One persistent connection
per `RuntimeClient` multiplexes everything, demultiplexed by a
caller-generated `call_id`:

```text
stream            { call_id, package, method, args, kwargs }
stream:item       { call_id, value }
stream:end        { call_id }
stream:error      { call_id, error }

bidi:start        { call_id, package, method, args, kwargs }
bidi:in           { call_id, item }
bidi:end_in       { call_id }
bidi:out          { call_id, value }
bidi:end          { call_id }
bidi:error        { call_id, error }

logs:subscribe    { filter?: <logger-name prefix> }
log               { level, name, message, timestamp }
logs:unsubscribe  {}
```

`RuntimeClient.remote` picks the wire automatically from the function:
plain `async def` → unary HTTP; `async def ... yield` → server-streaming
Socket.IO; same + a `Channel[T]` parameter → bidi. The call returns
`Unary[R]` / `Stream[R]` / `Bidi[I, R]`; each variant supports its
natural Python protocol (`await` or `async for`).

## Caller side

```python
from agentix import RuntimeClient
from agentix.myagent import run

async with RuntimeClient(sandbox.runtime_url) as c:
    result = await c.remote(run, instruction="…")
    # result: str — pyright infers from run's return type
```

`RuntimeClient.remote(fn, *args, **kwargs)`:

- Reads `fn.__module__` → routing key (the entry-point value's left side).
- Reads `fn.__name__` → method.
- Serialises args/kwargs via pydantic `TypeAdapter` driven by
  `inspect.signature(fn)`.
- Decodes the response into `fn`'s return type.

## Streaming methods

```python title="src/agentix/chatbot/__init__.py"
from collections.abc import AsyncIterator

async def chat(prompt: str) -> AsyncIterator[str]:
    proc = await asyncio.create_subprocess_exec("claude", "-p", prompt, ...)
    async for line in proc.stdout:
        yield line.decode()
```

```python
async for token in c.remote(chat, prompt="…"):
    print(token, end="")
```

Items flow over Socket.IO `stream:item`. The stream closes with
`stream:end`; impl exceptions arrive as `stream:error` and the client
raises `RemoteCallError`. Already-yielded items are observed before
the raise.

## Bidirectional methods

```python
from agentix import Channel
from collections.abc import AsyncIterator

async def echo(messages: Channel[str]) -> AsyncIterator[str]:
    async for msg in messages:
        yield f"echo: {msg}"
```

```python
inbox: Channel[str] = Channel()

async def producer():
    for line in user_inputs():
        await inbox.send(line)
    await inbox.close()

async for reply in c.remote(echo, messages=inbox):
    print(reply)
```

Inputs flow as `bidi:in`; outputs as `bidi:out`.

## Runtime built-ins

Independent of any namespace:

| Endpoint              | Purpose                                                                                                                                              |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| `GET /health`         | Liveness.                                                                                                                                            |
| `POST /exec`          | Run a shell command. Body `{command, cwd?, env?, timeout?}`. SSE when `Accept: text/event-stream`; else JSON `{exit_code, stdout, stderr}`.          |
| `POST /upload`        | Multipart upload into `AGENTIX_UPLOAD_ROOT` (default `/workspace`).                                                                                  |
| `GET /download?path=` | Stream a file back.                                                                                                                                  |
| `GET /namespaces`     | List loaded namespaces.                                                                                                                              |

`RuntimeClient.run / upload / download / namespaces` are typed Python
helpers over these.

## Minimal recipe

```text
my-namespace/
├── pyproject.toml          # [project.entry-points."agentix.namespace"]
└── src/agentix/<name>/
    └── __init__.py         # async def ...
```

`pip install ./my-namespace` makes it discoverable. `primitives/bash`
and `primitives/files` are working in-tree examples.
